<div align="center">

<img src="https://capsule-render.vercel.app/api?type=rect&height=48&text=%F0%9F%94%A7+GETTING+STARTED&fontSize=18&color=0:0d2137,100:050d1a&fontColor=00d4ff&animation=fadeIn" width="100%"/>

</div>

<br/>

### Prerequisites

```bash
# Required
Java 17+
Node.js 18+
Maven 3.8+
MySQL 8+
Docker (optional)
```

### Local Setup

```bash
# 1. Clone the repository
git clone https://github.com/ankit-7i/SkyLine.git
cd SkyLine

# 2. Configure the backend datasource
# Spring Boot reads these as environment variables (no application.properties
# is committed to the repo, so set your own or pass them via -D flags)
export SPRING_DATASOURCE_URL=jdbc:mysql://localhost:3306/skyline
export SPRING_DATASOURCE_USERNAME=root
export SPRING_DATASOURCE_PASSWORD=yourpassword

# 3. Run the backend
cd backend
mvn spring-boot:run
# API      → http://localhost:8080
# Swagger  → http://localhost:8080/swagger-ui/index.html

# 4. Run the frontend (new terminal)
cd frontend
npm install
npm start
# App → http://localhost:3000
```

### Docker Setup

```bash
# Backend
cd backend
docker build -t skyline-backend .
docker run -p 8080:8080 skyline-backend

# Frontend (point it at your running backend)
cd ../frontend
docker build -t skyline-frontend --build-arg REACT_APP_API_URL=http://localhost:8080 .
docker run -p 3000:80 skyline-frontend
```

> 💡 The API allows CORS from `http://localhost:3000` by default — update the `@CrossOrigin` origins in each controller if your frontend runs elsewhere.

<table>

<td width="50%" valign="top">

**Backend Dockerfile**
```dockerfile
FROM maven:3.9.4-eclipse-temurin-17 AS build
WORKDIR /app
COPY pom.xml .
COPY src ./src
RUN mvn clean install -DskipTests

FROM eclipse-temurin:17-jre-alpine
WORKDIR /app
COPY --from=build /app/target/*.jar app.jar
EXPOSE 8080
ENTRYPOINT ["java", \
  "-Dspring.profiles.active=prod", \
  "-jar", "app.jar"]
```

</td>
<td width="50%" valign="top">

**Frontend Dockerfile**
```dockerfile
FROM node:18-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
ARG REACT_APP_API_URL
ENV REACT_APP_API_URL=$REACT_APP_API_URL
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/build \
     /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

</td>
</tr>
