\# ✈ SkyLine - Flight Reservation System



A full stack flight reservation web application.



\## Tech Stack

\- \*\*Frontend:\*\* React 18, CSS3

\- \*\*Backend:\*\* Spring Boot 3, Java 17, JWT

\- \*\*Database:\*\* MySQL



\## Structure

\- `/backend` — Spring Boot REST API

\- `/frontend` — React frontend



\## Run Locally



\### Backend

cd backend

mvn spring-boot:run



\### Frontend

cd frontend

npm install

npm start
